Quick Send Mobile App

Quick Send Mobile App

Role

Design

Tools

Adobe Photoshop, Adobe Illustrator

Summary

Mobile app design for digitizing paper processes.

The Challenge

Melaleuca's business model is built on direct marketing and sales which means lots of paperwork for new sales reps. I was tasked to design an interface for a mobile application that would allow the various paperwork tasks be captured and transmitted digitally to headquarters. The app needed to work well with little data or limited connectivity and needed to be branded and familiar enough to the user for them to feel confident using it to make submissions.

The Solution

I worked closely with the creative director and app development team to create the initial user flow. The app was initially busy, so in a following iteration I simplified the design using more understandable icons and reduced several unecessary steps from the flow. The end result was an intuitive and useful application that reduced paper waste and sped up data entry.
